 <span data-ttu-id="a070f-158">Par exemple, supposons que trois prédictions soient effectuées pour un état cible donné et que les probabilités de chaque prédiction soient égales à 0,05, 0,15 et 0,8.</span><span class="sxs-lookup"><span data-stu-id="a070f-158">For example, suppose three predictions are made for a particular target state, and the probabilities of each prediction are 0.05, 0.15, and 0.8.</span></span> <span data-ttu-id="a070f-159">Si vous définissez la valeur 0,5 pour le seuil, une seule prédiction est comptabilisée comme correcte.</span><span class="sxs-lookup"><span data-stu-id="a070f-159">If you set the threshold to 0.5, only one prediction is counted as being correct.</span></span> <span data-ttu-id="a070f-160">Si vous affectez à **Seuil** **cible** la valeur 0,10, deux prédictions sont comptabilisées comme correctes.</span><span class="sxs-lookup"><span data-stu-id="a070f-160">If you set **Target** **Threshold** to 0.10, two predictions are counted as being correct.</span></span>
