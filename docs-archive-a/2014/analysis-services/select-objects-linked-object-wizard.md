# <a name="select-objects-linked-object-wizard"></a>Sélectionner les objets (Assistant Objet lié)
  Utilisez la page **Sélectionner des objets** pour sélectionner les objets qui feront l'objet d'une liaison ou qui seront importés.  
  
> [!NOTE]  
>  Les dimensions et les groupes de mesures (y compris les mesures associées) sont liés. Les actions, les indicateurs de performance clés (KPI) et les calculs sont importés.  
  
## <a name="options"></a>Options  
 **Objets disponibles**  
 Sélectionnez les objets à lier ou importer.  
  
> [!NOTE]  
>  Seules les dimensions sont disponibles si un cube n’est pas sélectionné dans [!INCLUDE[ssBIDevStudioFull](../includes/ssbidevstudiofull-md.md)] à l’ouverture de l’Assistant Objet lié.
