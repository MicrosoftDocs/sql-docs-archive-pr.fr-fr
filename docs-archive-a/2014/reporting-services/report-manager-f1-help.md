# <a name="report-manager-f1-help"></a>Aide F1 du Gestionnaire de rapports
  Les rubriques suivantes fournissent de l’aide au niveau de la page pour [!INCLUDE[ssNoVersion](../includes/ssnoversion-md.md)] [!INCLUDE[ssRSnoversion](../includes/ssrsnoversion-md.md)] Gestionnaire de rapports. Il existe une rubrique pour chaque page de l'application. Certaines pages risquent de ne pas être disponibles. Les paramètres de sécurité déterminent les pages auxquelles vous avez accès. Pour accéder à ces rubriques à partir du Gestionnaire de rapports, cliquez sur **Aide** dans la barre d'outils principale.
