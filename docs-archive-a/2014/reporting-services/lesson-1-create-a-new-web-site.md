# <a name="lesson-1-create-a-new-web-site"></a>Leçon 1 : créer un nouveau site Web
  Dans cette leçon, vous allez apprendre à créer un projet de site web en utilisant le modèle de site web ASP.NET pour Visual C#.  
  
### <a name="to-create-a-new-website"></a>Pour créer un nouveau site Web  
  
1.  Ouvrez Microsoft Visual Studio 2010 ou Microsoft Visual Studio 2012.  
  
2.  Dans le menu **Fichier** , pointez sur **Nouveau**, puis sélectionnez **Site Web**.  
  
3.  Dans la boîte de dialogue **nouveau site Web** , dans le volet **modèles installés** , sélectionnez **Visual C#** , puis choisissez **site Web ASP.net**.  
  
4.  Dans la zone **emplacement** , spécifiez un répertoire de projet et cliquez sur **OK**.  
  
 Le projet de site Web s'ouvre.
