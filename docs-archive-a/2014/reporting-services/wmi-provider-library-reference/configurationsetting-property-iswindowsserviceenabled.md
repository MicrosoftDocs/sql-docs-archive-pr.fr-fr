# <a name="iswindowsserviceenabled-property-wmi-msreportserver_configurationsetting"></a>Propriété IsWindowsServiceEnabled (WMI MSReportServer_ConfigurationSetting)
  Indique si le service Windows Report Server est activé. Lecture seule.  
  
## <a name="syntax"></a>Syntaxe  
  
```vb  
Public Dim IsWindowsServiceEnabled As Boolean  
```  
  
```csharp  
public boolean IsWindowsServiceEnabled;  
```  
  
## <a name="property-values"></a>Valeurs de la propriété  
 Valeur **booléenne** en lecture seule. La valeur `true` indique que le service Windows Report Server est activé.
