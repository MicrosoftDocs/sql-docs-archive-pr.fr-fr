# <a name="virtualdirectoryreportmanager-property-wmi-msreportserver_configurationsetting"></a>Propriété VirtualDirectoryReportManager (WMI MSReportServer_ConfigurationSetting)
  Retourne le répertoire virtuel de l'application Gestionnaire de rapports. Lecture seule.  
  
## <a name="syntax"></a>Syntaxe  
  
```vb  
Public Dim VirtualDirectoryReportManager As String  
```  
  
```csharp  
public string VirtualDirectoryReportManager;  
```  
  
## <a name="property-values"></a>Valeurs de la propriété  
 Objet `string` qui contient le répertoire virtuel.
