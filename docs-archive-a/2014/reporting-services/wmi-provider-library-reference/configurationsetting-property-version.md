# <a name="version-property--wmi-msreportserver_configurationsetting"></a>Propriété Version (WMI MSReportServer_ConfigurationSetting)
  Retourne la version du serveur de rapports au format Major.Minor.Build.Revision. Lecture seule.  
  
## <a name="syntax"></a>Syntaxe  
  
```vb  
Public Dim Version As String  
```  
  
```csharp  
public string Version;  
```  
  
## <a name="property-value"></a>Valeur de propriété  
 Valeur `string` qui contient la version du serveur de rapports.
