# <a name="copy-query-results-visual-database-tools"></a>Copier les résultats d'une requête (Visual Database Tools)
  Vous pouvez copier le jeu de résultats sous la forme de texte séparé par des tabulations à partir du volet **Résultats** . Vous pouvez copier le jeu de résultats complet ou plusieurs lignes ou cellules contiguës ou non contiguës.  
  
### <a name="to-copy-result-set-text-from-the-results-pane"></a>Pour copier le texte du jeu de résultats à partir du volet Résultats  
  
1.  Exécutez la requête ou la vue contenant les données que vous souhaitez copier.  
  
2.  Pour sélectionner le jeu de résultats complet, dans le volet **Résultats** , cliquez sur la cellule vide en haut à gauche.  
  
3.  Pour sélectionner des lignes :  
  
    -   Sélectionnez une ligne en cliquant sur la zone à gauche.  
  
    -   Sélectionnez plusieurs lignes contiguës en choisissant une ligne puis en faisant glisser la souris pour sélectionner les autres lignes.  
  
    -   Sélectionnez des lignes non contiguës en choisissant une ligne, puis en maintenant la touche Ctrl enfoncée et en cliquant sur les autres lignes.  
  
     Les lignes sélectionnées sont mises en surbrillance.  
  
4.  Pour sélectionner des cellules :  
  
    -   Sélectionnez une seule cellule en cliquant sur la valeur.  
  
    -   Sélectionnez plusieurs cellules en faisant glisser la souris vers des cellules adjacentes.  
  
    -   Sélectionnez des cellules non contiguës en choisissant une seule cellule, puis en maintenant la touche Ctrl enfoncée et en cliquant sur les autres cellules.  
  
     Les cellules sélectionnées sont mises en surbrillance.  
  
5.  Cliquez avec le bouton droit, puis cliquez sur **Copier**.
