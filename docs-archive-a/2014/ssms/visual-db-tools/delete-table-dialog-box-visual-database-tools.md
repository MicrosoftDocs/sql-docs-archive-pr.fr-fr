# <a name="delete-table-dialog-box-visual-database-tools"></a>Boîte de dialogue Supprimer une table (Visual Database Tools)
  Cette boîte de dialogue permet de spécifier la table dans laquelle des enregistrements sont à supprimer. Elle apparaît si plusieurs tables sont affichées dans le volet Schéma lorsque vous remplacez le type d'une requête par une requête DELETE.  
  
 Sélectionnez la table dans laquelle les enregistrements doivent être supprimés, puis choisissez **OK**.  
  
> [!NOTE]  
>  Une requête DELETE supprime de la table des lignes entières. Si vous souhaitez effacer des valeurs dans des colonnes de données individuelles, utilisez une requête UPDATE. Si vous souhaitez supprimer des colonnes dans une définition de table ou supprimer la table elle-même, utilisez le Concepteur de tables ou les outils de design de votre base de données.
